# Telegram bot for Bitconnect lending

**Category:** Telegram Bot<br/>
**Tools:** Node.js(node-telegram-bot-api,cheerio,request)<br/>
**BotName:** @BCCRoyalty_bot<br/>

## Usage and Readme

### Bot can:

* **/help** for all usable command
* **/royalty** give to you statistic about persent of royalty(today,yesterday,tomorow)
* **/payments <your deposit now> <day count> <count of your deposits more than 1010$>** You can calculate amount of your deposit in future after (count) of days
